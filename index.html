About

At-A-Glance is a single page web app that that simplifies the process of getting updated on information that is periodically checked throughout the day. Rather than individually checking separate apps or sites for weather, dates, news, and finance, this web app summarizes it all in one screen. For more detailed information, simply tap the desired widget.

This web app will be periodically updated with new features and widgets. Planned features include: a todo list widget, customizable themes, guided tour for new users, widget rearrangement via drag and drop, and hourly forecasts for the weather widget.
